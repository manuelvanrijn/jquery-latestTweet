jQuery.latestTweet

On this page you'll find a few examples that demonstrate the jQuery.latestTweet plugin

Plugin defaults

$.fn.latestTweet.defaults = {
  formatLinks: true,          // format url's to a clickable link
  formatMentions: true,       // format @username to a link to the users profile
  formatHashes: true,         // format #tag to a link to the twitter tag search
  callback: function() {}     // function that will be called after the latest tweet has been filled in.
};

Example 1: Get latest tweet for @bbcnews

Tweet

Code

$(function() {
  $('#example1 p').latestTweet('bbcnews');
});

Example 2: Get latest tweet without formatted hashes, url's and mentions

Tweet

Code

$(function() {
  $('#example2 p').latestTweet('bbcnews', {
    formatLinks: false,
    formatMentions: false,
    formatHashes: false
  });
});

Example 3: Get latest tweet with callback on finish

press the button to start

Tweet

Code

$(function() {
  $('#example3start').click(function(e) {
    e.preventDefault();
    $("#log").text('start getting latest tweet');
    $('#example3 p').latestTweet('foxnews', {
      callback: function() {
        $("#log").text('finished!');
      }
    });
  });
});
